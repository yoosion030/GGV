# [GSM Golden Village 🍿](https://gsm-golden-village.vercel.app/)

### Description

GGV는 해외 영화 소개 서비스 입니다. 사용자 친화적인 UI/UX로 간편하게 영화를 조회할 수 있습니다.

![](public/MockUp2.png)
![](public/MockUp.png)

### Get Start

```
$ git clone https://github.com/yoosion030/GGV.git
$ yarn install
$ yarn dev
```

### Use Skills

```
- Next.js + TypeScript
- axios
- jotai
- React-hook-form
- Emotion
- IntersectionObserver
```
